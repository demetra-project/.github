# Demetra
Demetra is a project that merges artificial intelligence and advanced technologies to protect certain areas, such as national parks and natural reserves. It focuses on detecting fires, to prevent environmental disasters, and on identifying waste using computer vision algorithms. Through real-time communication with a mobile app, the system ensures quick responses and efficient environmental monitoring.

<br>

## 📦 Main components
- Drone frame
- LiPo Battery
- Power Distribution Board (PDB)
- Brushless Motors + Propellers
- STEVAL-FCU01v2 (flight controller)
- X-NUCLEO-IKS4A1 (for environmental monitoring)
- GPS Module (for location tracking)
- Raspberry Pi 5(for object detection algorithms)
- Camera

<br>

## ⚙️ Features
- Air quality analysis to detect fires
- Visual waste recognition (through object detectiom algorithms)
- Real-time video feed for live surveillance
- GPS-based waste location tracking

<br>

## 🌱 Innovation and Sustainability
Demetra improves environmental surveillance using drones instead of traditional, polluting vehicles. This approach reduces resource waste and helps managing protected areas more efficiently, without the need for human intervention.
